About

Steamize is a Python-based web application using Django Framework. It makes use of Steam's Web API to extract data and information, given the account ID. It makes sure the API requests are called asynchronously to provide full optimization. The data is formatted into a structured JSON schema, where it is stored in a Mongo Database for easy retrieval by the Steamize app.
